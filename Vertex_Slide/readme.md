Vertex Slide Add-on for Blender
Version: 1.5

---------------------------------------------------

Installation:
1. Open Blender.
2. Go to Edit > Preferences > Add-ons > Install.
3. Select this .zip file and click 'Install Add-on'.
4. After installation, search for 'Vertex Slide' in the Add-ons list and enable it.

Usage:
1. Open the tab menu and click "1. Mark Edge" to select the reference edge.
2. Select the vertices you want to slide.
3. Click "2. Slide Vertices" and drag the slider to adjust.
4. Release to confirm the slide operation.

Supported Blender Versions:
Blender 4.1 and above.
